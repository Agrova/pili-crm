# Module: catalog

**Zone:** Reference data — product catalogue and supplier directory.

## Entities
- `Product` — tool item with characteristics, category, weights (declared / actual), photos
- `Supplier` — vendor (Lee Valley, Veritas, Lie-Nielsen, etc.)
- `ProductAttribute` — key/value attributes per product

## External Connectors
- Supplier website scraping (future)

## Dependencies
- Depends on: _none_
- Depended on by: `orders`, `procurement`, `pricing`, `communications`
