# Glossary

- Заказ клиента — заказ, который оформляет конечный покупатель.
- Закупка — заказ товара у поставщика или магазина-источника.
- Поступление на склад — подтверждённый факт прибытия товара на склад.
- Фактический вес — вес, подтверждённый складом или взвешиванием.
- Расчётный вес — вес, который используется для расчёта логистики и цены.
- Трек-номер — идентификатор отправления, используемый для связывания событий доставки.
- Purchase type — тип закупки: retail (розница) или manufacturer (от производителя). Определяет ветку формулы pricing.
- Pricing exchange rate — расчётный курс валюты, используемый в формуле pricing. Может включать наценку. Владелец — модуль pricing.
- Bank exchange rate — фактический курс, по которому реально менялись деньги. Из банковской выписки. Владелец — модуль finance.
- Planned / Actual — два слоя расчёта прибыли: planned (до продажи, по формуле pricing) и actual (после всех фактов, из finance).
- Derivation rule — правило автоматического вычисления статуса заказа по статусам его позиций.
- Форвардер — посредник в США, получающий товар от поставщика перед отправкой в Москву.
- Intermediary fee — фиксированная сумма в RUB за услуги посредника (например, Бисембай) по закупке у производителя.
- MCP-tool — инструмент MCP-сервера, через который Claude Cowork взаимодействует с PostgreSQL.
- Cowork — Claude Cowork, десктопный интерфейс оператора магазина.
- Prompt Factory — чат "Prompt Factory for Claude Code", через который готовятся промты для реализации задач.
